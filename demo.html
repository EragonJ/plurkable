Plurkableー輕量級 jQuery plugin，可以輕易的與 Web application 做整合。提供比原本 Plurk Widget 更加高度客制化的功能，你能夠依自己的需求依參數來調整 Layout，未來將 support 多人帳號混用。下載

你只需要做以下設定即可： $(document).ready(function() { $("#plurk").plurkable({ num : 6, // 顯示噗數（最多 20 則） username : 'EragonJ', // 使用者帳號 date_format : 'default', // 時間格式 show_link : true, // 是否要顯示連結 show_date : true // 是否要顯示日期 }); });
